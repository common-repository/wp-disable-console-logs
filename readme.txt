WordPress plugin to prevent javascript from loggin to the console 

== Description ==

This WordPress Plugin prevents javascript console logs unless you are logged in as an administrator. Great for added security and hiding any activity you might have going on behind the scenes. 

* No setup required!

== Installation ==

Upload the "wp-disable-console-logs" folder to your wp-content/plugins directory, then Activate it.
